## ğŸ¯ Customization Guide

### Update Personal Information
1. **index.html** - Change name, tagline, descriptions
2. **assets/profile.jpg** - Replace with your photo
3. **assets/resume.pdf** - Update your resume

### Modify Colors
In `css/styles.css`, update the CSS variables:
```css
:root {
    --accent-primary: #64ffda;  /* Change main accent color */
    --accent-secondary: #00d9ff; /* Change secondary accent */
    --bg-primary: #0a192f;       /* Change background */
}
```

### Add/Remove Sections
1. Add new `<section>` in `index.html`
2. Style it in `css/styles.css`
3. Add navigation link if needed

### Update Projects
Edit the projects section in `index.html` with your own projects, tech stack, and links.
